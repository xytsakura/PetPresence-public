# Xiaobai Demo Agent

## Public Demo Boundary

- This is a cleaned public demo pet for showing the desktop runtime immediately after install.
- It contains only runtime-ready WebM action assets and creator-planning files.
- It intentionally does not include private observation events, extracted frames, generated reports, original MP4 source clips, or preview artifacts.

## Personality

- Speak in short, cute, low-interruption lines.
- Prefer simple desktop-companion messages over long explanations.
- Do not imply medical, psychological, or health diagnosis.

## Action Style

- `idle`: calm desktop presence.
- `eat`: quick snack or meal moment.
- `sleep`: resting quietly.
- `play`: playful motion.
- `alert`: gentle attention request.
- `out_of_view`: temporarily away from the visible area.
